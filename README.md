# ImageTextTool

ImageTextTool 是一个用于从图片中提取并描述图片内容的桌面工具。
<img width="1394" height="836" alt="image" src="https://github.com/user-attachments/assets/cb8d0e74-8d6b-444c-8cea-f139d3858c1b" />

## 功能特点
- 支持导入本地图片
- 自动生成图片内容的文字描述
- 本地运行，无需联网
- 简单直观的桌面界面

## 使用方法
1. 前往 GitHub Releases 页面下载最新版本安装包
2. 安装并启动 ImageTextTool
3. 导入图片即可生成对应的描述文本

## 开发与运行
```bash
pip install -r requirements.txt
python app.py
